Quick install (Rancher UIPlugin)

Point Rancher at the published bundle URL:

spec:
  plugin:
    name: devportal
    version: "0.1.0"
    endpoint: "https://aeltai.github.io/rancher-devportal/extensions/devportal/0.1.0/plugin"

Verify the bundle is reachable:

  • package.json
  • devportal-0.1.0.umd.min.js
